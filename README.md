## 注意事項
- 本專案之資料庫使用Azure雲端資料庫系統，並且使用免費帳號。因此免費額度結束後便無法使用，專案也無法啟動。
- 某些檔案大小過大，因此本專案的Repository購買了Git LFS Data(容量擴充)。若擴充的時效過了，則無法完整下載本專案。
